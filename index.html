Use commands to perform instructions

  • Write file

    Enter '/write/(fileName)/(fileContent)' after baseURL to write fileContent into fileName

    for e.g- /write/intro.txt/My Name is Mousam

  • Read file

    Enter '/read/(fileName)' after baseURL to read content of fileName

    for e.g- /read/intro.txt

  • Rename file

    Enter '/rename/(oldFileName)/(newFileName)' after baseURL to rename oldFileName to newFileName

    for e.g- /rename/intro.txt/introduction.txt

  • Delete file

    Enter '/remove/(fileName)' after baseURL to delete the file fileName

    for e.g- /remove/introduction.txt
